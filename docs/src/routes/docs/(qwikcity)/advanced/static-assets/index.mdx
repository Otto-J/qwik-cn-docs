# 静态资源

使用 `/public` 文件夹存储所有静态资源，如图片、字体等。

所有静态资源都从您的服务器根目录 (`/`，例如 `https://example.com/`) 文件夹中提供，并在路由下叠加（路由具有优先级）。

```bash
public/
├── favicon.ico           # https://example.com/favicon.ico
└── images/
    └── logo.png          # https://example.com/images/logo.png
```
