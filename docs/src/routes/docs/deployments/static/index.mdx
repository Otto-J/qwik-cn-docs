# 静态网站适配器

Qwik 的静态网站适配器可以帮助生成静态的 HTML 文件，可以轻松部署到任何云服务提供商。

## 安装

要集成 `static-site` 适配器，使用 `add` 命令：

```shell
npm run qwik add static
```

上述命令将在项目根目录下创建一个名为 `adapters/static/vite.config.ts` 的目录，并包含以下代码。

```tsx file="apdaters/static/vite.config.ts"
import { extendConfig } from '@builder.io/qwik-city/vite';
import baseConfig from '../../vite.config';

export default extendConfig(baseConfig, () => {
  return {
    build: {
      ssr: true,
      rollupOptions: {
        input: ['@qwik-city-plan'],
      },
    },
    plugins: [
      staticAdapter({
        origin: 'https://yoursite.qwik.dev',
      }),
    ],
  };
});
```

> 请记得将该文件中的 `origin` 更改为您的域名。

现在，您可以在使用 `Qwik` 的丰富生态系统、基于文件的路由等功能的同时生成静态网站。
