# 欢迎使用 Qwik

Qwik 是一种新型网络框架，可提供即时加载的网络应用程序，无论其大小或复杂程度如何。您的网站和应用程序只需约 1kb 的 JS 即可启动（无论应用程序的复杂程度如何），并可实现稳定的性能。

## Qwik 目标

<div class="card-grid">
  <div class="card">
    <h3>通用型</h3>
    <p>Qwik 可以用语构建任何类型的网站或者应用程序</p>
  </div>
  <div class="card">
    <h3>即时启动</h3>
    <p>与其他框架不同，Qwik 是可恢复的，这意味着 Qwik 应用程序不需要水合。这使得 Qwik 应用程序具有即时交互性，无论大小或复杂性如何。</p>
  </div>
  <div class="card">
    <h3>针对速度进行了优化</h3>
    <p>Qwik 具有前所未有的性能，即使在移动设备上也能提供亚秒级的整页加载。Qwik 通过提供纯 HTML 并仅根据需要增量加载 JS 来实现这一点。</p>
  </div>
</div>
