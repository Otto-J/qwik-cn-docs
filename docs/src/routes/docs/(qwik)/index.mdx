# Qwik 欢迎

Qwik 是一种新型的网络框架，能够为任何规模或复杂程度的应用程序提供即时加载功能。您的网站和应用程序可以在大约 1kb 的 JS 中启动（无论应用的复杂度如何），并在大规模部署时实现一致性的性能。

## 为何选择 Qwik？

<div class="card-grid">
  <div class="card">
    <h3>通用型</h3>
    <p>Qwik 可以用于构建任何类型的网站或应用程序。</p>
  </div>
  <div class="card">
    <h3>即时互动性</h3>
    <p>
      Qwik 应用程序可以即时工作，无需任何延迟，无论其规模或复杂程度，都不需要任何预加载。
    </p>
  </div>
  <div class="card">
    <h3>JavaScript 流式传输</h3>
    <p>
      Qwik 提供了在用户主动选择时提供纯 HTML 并执行 JavaScript 的功能，即使在移动设备上也可以实现次秒级的全页加载。
    </p>
  </div>
</div>
