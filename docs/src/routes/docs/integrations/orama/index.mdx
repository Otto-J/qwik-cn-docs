# Orama

[Orama](https://oramasearch.com/) 是一个快速、功能齐全的全文搜索引擎，完全使用 TypeScript 编写，没有任何依赖。

Orama 可以在 Qwik 的客户端或者使用 `routeLoader$`、`routeAction$` 和 `server$` 函数在服务器端使用。

将 Orama 添加到 Qwik 的最简单方法是使用 `pnpm qwik add orama` 命令。这将安装所需的依赖项，并创建一个新的公共路由 `/src/routes/orama` 来展示 Orama 的集成。

```bash
npm run qwik add orama
```

> Orama 被设计为在任何运行时上工作，并且没有任何依赖。唯一的要求是一个 JavaScript 运行时。

更多信息，请查阅 [Orama 文档](https://docs.oramasearch.com/)。
