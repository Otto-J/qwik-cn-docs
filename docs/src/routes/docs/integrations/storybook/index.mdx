# Storybook

[Storybook](https://storybook.js.org/) 是一个用于在隔离环境中构建 UI 组件和页面的前端工作坊。成千上万的团队使用它进行 UI 开发、测试和文档编写。它是开源且免费的。

自 7.0 版本以来，Storybook 原生支持 Vite，这意味着它也对 Qwik 有着一流的支持。

## 使用方法

您可以将 Storybook 添加到现有的应用程序或库中使用：

```shell
npm run qwik add storybook
```

上述命令会安装必要的依赖项并添加一个示例组件和故事。

现在，您可以启动 Storybook 仪表板：

```shell
npm run storybook
```
