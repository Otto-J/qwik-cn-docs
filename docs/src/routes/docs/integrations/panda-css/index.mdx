# Panda CSS

Panda 是一个样式引擎，以类型安全和可读的方式生成样式原语，用于编写原子 CSS 和配方。
[Panda CSS 网站](https://panda-css.com/)

## 使用方法

您可以通过使用以下 Qwik 脚本轻松添加 Panda：

```shell
npm run qwik add pandacss
```

上述命令会更新您的应用程序，并添加所需的依赖项。

它还会向您的项目文件夹添加新文件：

- `postcss.config.js`
- `panda.config.js`
- `.vscode/settings.json`

并修改您的 `src/global.css` 文件以包含以下内容：

```css title="src/global.css"

# global.css 文件

@layer reset, base, tokens, recipes, utilities;

...内容...

```
