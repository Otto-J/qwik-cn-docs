# Cypress 

使用 [Cypress](https://www.cypress.io)，您可以轻松为现代 Web 应用创建测试，通过可视化调试它们，并在持续集成构建中自动运行它们。


## 组件测试

Cypress 可以通过 [cypress-ct-qwik](https://github.com/qwikifiers/cypress-qwik) 社区插件，
帮助您在隔离环境中测试 Qwik 组件。




## 使用方法

您可以通过使用以下 Qwik 起始脚本轻松添加 Cypress：

```shell
npm run qwik add cypress
```

此命令会将 `cypress` 和 `cypress-ct-qwik` 添加到您的依赖项中，
将测试命令追加到您的 `package.json` 脚本部分，并创建运行 Cypress 组件测试所需的配置文件。


有关更多信息，请查阅 [Cypress 文档](https://docs.cypress.io/guides/overview/why-cypress) 和 [cypress-qwik 文档](https://github.com/qwikifiers/cypress-qwik)。
