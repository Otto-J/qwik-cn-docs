# 集成

Qwik CLI 提供了一些集成功能，使得连接到您的工具和服务变得更加容易。

创建 Qwik City 应用程序后，运行以下命令以添加新的集成：

```shell
npm run qwik add
```

它会提示您选择要添加的集成。选择集成后，它将被添加到您的应用程序中，您可以开始使用它。
