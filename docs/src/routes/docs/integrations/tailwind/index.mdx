# Tailwind

Tailwind 是一个 CSS 框架，它为我们提供了单一目的的实用类，这些类在很大程度上是有偏见的，并且可以帮助我们在标记或 `.js/.jsx/.ts/.tsx/.mdx` 文件中设计我们的网页。[Tailwindcss 网站](https://tailwindcss.com/)

## 使用方法

您可以使用以下 Qwik 脚本轻松添加 Tailwind：

```shell
npm run qwik add tailwind
```

上述命令会更新您的应用程序，并添加所需的依赖项。

它还会向您的项目文件夹添加新文件：

- `postcss.config.js`
- `tailwind.config.js`
- `.vscode/settings.json`

并修改您的 `src/global.css` 文件以包含以下内容：

```css title="src/global.css"

# global.css 文件

@tailwind base;
@tailwind components;
@tailwind utilities;

...stuff...

```
