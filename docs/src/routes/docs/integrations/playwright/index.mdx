# Playwright

Playwright 可以在支持所有现代渲染引擎的现代 Web 应用中进行可靠的端到端测试，并且可以跨平台使用。[Playwright 网站](https://playwright.dev/)。

## 使用方法

您可以通过使用以下 Qwik 脚本轻松添加 Playwright：

```shell
npm run qwik add playwright
```

上述命令会将 Playwright 添加到您的依赖项中，在 `package.json` 的 scripts 部分追加一个 `test.e2e` 命令，并在应用程序根目录下创建一个默认配置文件 `playwright.config.ts`。
它还会在新创建的 `tests` 文件夹中添加一个名为 `example.spec.ts` 的文件，其中包含一些简单的测试用例。

有关更多信息，请查阅 [Playwright 文档](https://playwright.dev/docs/intro)。
